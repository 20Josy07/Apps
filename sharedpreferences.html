1. Navegación (Bottom Navigation + Fragments)

Concepto: El Bottom Navigation cambia los "Fragmentos" dentro de un contenedor en la pantalla principal.

res/menu/nav_menu.xml
<menu xmlns:android="http://schemas.android.com/apk/res/android">
    <item
        android:id="@+id/nav_home"
        android:icon="@drawable/ic_home"
        android:title="Inicio" />
    <item
        android:id="@+id/nav_config"
        android:icon="@drawable/ic_settings"
        android:title="Configuración" />
</menu>
res/layout/activity_main.xml
<RelativeLayout xmlns:android="..." ...>
    <!-- Aquí se cargarán los fragmentos -->
    <FrameLayout
        android:id="@+id/contenedor_fragmentos"
        android:layout_width="match_parent"
        android:layout_height="match_parent"
        android:layout_above="@id/bottom_nav" />

    <com.google.android.material.bottomnavigation.BottomNavigationView
        android:id="@+id/bottom_nav"
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        android:layout_alignParentBottom="true"
        app:menu="@menu/nav_menu" />
</RelativeLayout>
MainActivity.java
public class MainActivity extends AppCompatActivity {
    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        BottomNavigationView nav = findViewById(R.id.bottom_nav);
        
        // Cargar fragmento por defecto al abrir la app
        cambiarFragmento(new HomeFragment());

        nav.setOnItemSelectedListener(item -> {
            if (item.getItemId() == R.id.nav_home) {
                cambiarFragmento(new HomeFragment());
            } else if (item.getItemId() == R.id.nav_config) {
                cambiarFragmento(new ConfigFragment());
            }
            return true;
        });
    }

    private void cambiarFragmento(Fragment f) {
        getSupportFragmentManager().beginTransaction()
            .replace(R.id.contenedor_fragmentos, f)
            .commit();
    }
}

2. SharedPreferences (Persistencia de Datos)

Úsalo para guardar el nombre del usuario, si aceptó términos, o su puntaje.
SharedPrefManager.java (Clase de Utilidad)
public class SharedPrefManager {
    private SharedPreferences sharedPreferences;
    private SharedPreferences.Editor editor;

    public SharedPrefManager(Context context) {
        sharedPreferences = context.getSharedPreferences("MiAppPrefs", Context.MODE_PRIVATE);
        editor = sharedPreferences.edit();
    }

    public void guardarNombre(String nombre) {
        editor.putString("nombre_key", nombre);
        editor.apply(); // Guarda los cambios
    }

    public String obtenerNombre() {
        // Devuelve "Invitado" si no hay nada guardado
        return sharedPreferences.getString("nombre_key", "Invitado");
    }
}

3. Imágenes, Internet y Círculos

CircleImageView recorta la foto automáticamente. Picasso la descarga de la web.

layout_fragment_perfil.xml
<de.hdodenhof.circleimageview.CircleImageView
    android:id="@+id/imagen_circular"
    android:layout_width="150dp"
    android:layout_height="150dp"
    android:src="@drawable/foto_placeholder"
    app:civ_border_width="2dp"
    app:civ_border_color="#3ddc84" />
FragmentPerfil.java
@Override
public void onViewCreated(View view, Bundle savedInstanceState) {
    super.onViewCreated(view, savedInstanceState);
    
    CircleImageView profileImg = view.findViewById(R.id.imagen_circular);
    String url = "https://i.imgur.com/7Yf7T.png"; // URL de internet

    // PICASSO: Descarga, ajusta y pone la imagen en el círculo
    Picasso.get()
        .load(url)
        .placeholder(R.drawable.cargando) // Imagen mientras carga
        .error(R.drawable.error)       // Imagen si falla el link
        .into(profileImg);
}
